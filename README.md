Project Context:
Overseer is a specialized management tool for Ark: Survival Evolved communities. It automates the tedious parts of server administration: tribe registration, private roster management, and channel infrastructure



How to Test the Bot:
Initial Setup: Run /setup to designate an admin role and a staff log channel.
Category Setup: Run /setup-category and select a category. This is where the bot will "spawn" private tribe headquarters.
Registration: Use /post-info to deploy the button interface.
The "Magic" Moment: Click [Create Tribe]. The bot will:
Collect your IGN and Xbox Gamertag via Modal.
Automatically create a private text channel in your designated category.
Sync your Discord nickname to the [Tribe] Name format.
Post a persistent management dashboard in the new channel.

Permission Justification:
Manage Channels & Roles: Required to create private tribe headquarters and handle member access automatically.
Manage Nicknames: Required for the automated tribe-tagging feature.
